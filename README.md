# Şifre Yöneticisi

Bu proje, kullanıcıların şifrelerini güvenli bir şekilde saklamalarını ve yönetmelerini sağlayan bir şifre yöneticisidir. Streamlit kullanılarak oluşturulmuş ve Python ile geliştirilmiştir.

## Özellikler
- Şifrelerinizi platform, e-posta ve güç seviyelerine göre saklama.
- Şifre gücünü değerlendirme (Güçlü, Orta, Zayıf).
- Şifrelerinizi AES tabanlı bir şifreleme ile koruma.
- Şifrelerinizi görüntüleme, ekleme ve silme seçenekleri.
- Kullanıcı dostu bir arayüz.

## Gereksinimler
- Python 3.7 veya daha üstü
- Gerekli Python kütüphaneleri:
  - `streamlit`
  - `cryptography`
  - `pandas`
